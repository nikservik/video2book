# Улучшения 

## Проектные pipelines 
Есть идея делать версии пайплайнов, адаптированные под проект. 
Тогда базовых пайплайнов будет мало. А проектные будут видны только внутри проекта и не будут засорять список.

- считать длительность аудио в уроке и показывать
- мобильная верстка
