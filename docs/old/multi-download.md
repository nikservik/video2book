# Фича: Проект из скачанных уроков

Нам нужно добавить возможность создавать проект целиком из ссылок на YouTube.

## На главной странице
- Добавляем кнопку "+ Проект с YouTube" рядом с кнопкой "+ Создать проект".
- При нажатии на кнопку «+ Проект с YouTube» переходим на новую страницу создания проекта CreateYtProjectView.

## CreateYtProjectView
- За образец берем CreateLessonView в режиме создания проекта.
- ProgressBar здесь не нужен.
- Первое поле – это название проекта.
- Второе - выбор пайплайна.
- Дальше, textarea на 20 строк. Это будет текстовый список уроков со ссылками.
    - Если строка начинается с #, то это название урока.
    - Следующая строка после названия — это ссылка на видео, которое нужно скачать.
    - Уроки могут быть разделены любым количеством пустых строк.

### Пример корректного списка уроков:
```
# Урок 1
https://www.youtube.com/watch?v=-PRaQKjyR5o
# Урок 2
https://www.youtube.com/watch?v=-PRaQKjyR5o

# Урок 3

https://www.youtube.com/watch?v=-PRaQKjyR5o



# Урок 4
https://www.youtube.com/watch?v=-PRaQKjyR5o
```

### Плейсхолдер для списка уроков:
```
# Урок 1
https://www.youtube.com/watch?v=...

# Урок 2
...
```

### Проверка корректности списка
- Для проверки просто пытаемся создать список уроков со ссылками. Если получилось, значит список корректный.
- Ссылки проверяем только на наличие 'https://' в начале и всё.

### Создание проекта
Если вы удается создать корректный список уроков и заполнить имя проекта и выбрать pipeline, то вы можете создать проект и все уроки для него. Инициализируем загрузки и дальше всё происходит уже на бэкэнде, как обычно.
